Syntax

Possible Syntaxes for this command. Type what you see exactly, but replace what is between <> with your own values.

>perm

Shows your permission level

>perm set @<​User><#>

Sets the Permissions of the user(s) @​mentioned

Options: 0 - Normal | 1 - Regular | 2 - Moderator | 3 - Admin

>perm set @<​Role><#>

Sets the Permissions of all users with the Role(s) @​mentioned

Options: 0 - Normal | 1 - Regular | 2 - Moderator | 3 - Admin

>perm all

Lists all permissions for the server
